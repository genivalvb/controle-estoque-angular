Interface Consumidora da API: ControleEstoqueAPI

Essa interface se comunica com uma API de Controle de Estoque para buscar e carregar informações sobre estoque
Há alguns objetos nesse API:
  • Reserve: Uma reserva cadastrada do que teremos que comprar no futuro
  • Prevision: Uma compra futura já estabelecida
  • Item: Os items cadatrados que são utilizados pelo comércio
  • Movement: A movimentação entre entrada e saída de items do estoque
  • Stock: O estoque propriamente dito.
